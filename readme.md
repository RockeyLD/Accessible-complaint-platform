# 吐槽列表（综合版）用户使用手册

适用文件：`查看吐槽.html`

## 1. 简介
“吐槽列表（综合版）”用于统一查看与管理本地保存的吐槽记录。数据只保存在当前浏览器的本地存储中，可上传到云端。

## 2. 打开方式
1) 直接用浏览器打开 https://rockeyld.github.io/Accessible-complaint-platform/
2) 建议使用常见现代浏览器（Chrome / Edge / Firefox）。

## 3. 页面结构
- 顶部：标题与简要说明。
- 管理面板：主题、字号、高对比度、语音辅助开关。
- 状态栏：显示加载、刷新、复制等提示信息。
- 历史记录区：搜索、返回上一页、刷新、清空所有，以及吐槽列表。

## 4. 常用操作

### 4.1 主题风格
在“主题风格”下拉框中选择配色（暖晨橙绿 / 海风蓝绿 / 日光米黄 / 清爽灰白）。系统会记住你的选择。

### 4.2 文字大小
拖动“文字大小”滑块即可放大或缩小全站字体。

### 4.3 高对比度
点击“开启高对比”切换高对比模式，提高可读性。再次点击可关闭。

### 4.5 搜索吐槽
在“搜索关键词...”输入框中输入关键词，列表实时过滤匹配的标题、内容或元信息。

### 4.6 刷新列表
点击“刷新”可重新读取本地数据并更新显示。

### 4.7 复制内容
在每条吐槽的“复制内容”按钮上点击，即可将“标题 + 内容”复制到剪贴板。

### 4.8 删除单条
点击“删除”按钮会弹出确认框，确认后删除该条记录。

### 4.9 清空所有
点击“清空所有”将删除所有本地吐槽记录（包含两个版本的数据），请谨慎操作。

### 4.10 tab键切换选项
按下“tab”键将在按钮间跳跃。

### 4.11 自动补全
如果听障吐槽中用户输入信息不完整，系统会询问并自动补全。

## 5. 附件支持
若某条吐槽包含附件：
- 图片会显示预览图。
- 音频/视频会显示播放器。
- 可点击“下载附件”保存到本地。
